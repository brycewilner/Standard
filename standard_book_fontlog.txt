Fontlog for Standard Book

This file provides detailed information on the Standard Book font software. This information should be distributed along with the Standard Book font and any derivative works.

Basic Font Information

Standard Book is a Unicode typeface family that supports all languages that use the Latin script and its variants, and could be expanded to support other scripts.
